## FLUX V1 Image Placement Guide

Use this folder path: `LogicStudio_Site/assets/flux-v1/`

### Required image files

1. Hero image
- File: `hero/flux-v1-hero.png`
- Recommended: `1600 x 1000` (aspect `8:5`)
- Minimum: `1200 x 750`

2. Flash workflow screenshot
- File: `screens/flash-page-preview.png`
- Recommended: `1600 x 1000` (aspect `8:5`)
- Minimum: `1280 x 800`

3. Optional section visuals
- Folder: `sections/`
- Keep aspect near `16:10` or `8:5` for consistent card fit.

### Notes
- The root `/index.html` page already points to these paths.
- If an image is missing, the page shows an in-card placeholder automatically.
